
# Bot de Figurinhas Pro

## Como usar localmente (no seu PC)
1. Instale as dependências:
   pip install -r requirements.txt

2. Rode o bot:
   python figurinhas_menu_bot.py

## Como usar no Render (online 24h)
1. Suba esses arquivos para um repositório no GitHub
2. Crie uma conta em https://render.com
3. Clique em "New > Web Service" e conecte com o GitHub
4. Use estas configurações:
   - Build command: pip install -r requirements.txt
   - Start command: python figurinhas_menu_bot.py
   - Environment: Python 3.x
